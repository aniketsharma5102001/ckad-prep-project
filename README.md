# CKAD Kubernetes Project 🚀

This project demonstrates core Kubernetes concepts using a real-world setup.

## 🔧 Features
- Namespace creation
- ConfigMap & Secret management
- Deployment with sidecar container
- Liveness & Readiness probes
- Shared volumes (emptyDir)
- ClusterIP Service
- NetworkPolicy (restricted access)

## 📦 Tech Stack
- Kubernetes
- Docker
- Nginx
- BusyBox

## 🚀 How to Run

```bash
kubectl apply -f .
